Get your Firebase config (free):
  1. Go to console.firebase.google.com → Add project → name it anything
  2. Click gear ⚙ → Project settings → Web icon </> → Register app → copy the 6 values below
  3. Build → Firestore Database → Create → Production mode → asia-south1
  4. Firestore → Rules tab → replace with: allow read, write: if request.auth != null; → Publish
  5. Build → Authentication → Get started → Email/Password → Enable
  6. Authentication → Settings → Authorised domains → Add your Netlify URL
